Globlin — A Free Logotype

This one was inspired by and old Fontomas release↗. I love the blobby, lava-lamp-esque shape of that font. But unfortunately, it seems to no longer exist. So I set out to create somethign similar. My version was created using a set of three circular shapes: one big circle, one small circle, and one small circle sheered 25° to create angles. These shapes were then moved around a grid and rotated to create letterforms.

At initial release, it only includes A-Z in lowercase in two weights. I'd like to eventually add numbers and punctuation. Feel free to take this idea and further iterate! I've included an Illustrator file in the download that includes my grid.
