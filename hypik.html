Hypik — A Free Font

This display font is an homage to Y2K aesthetics and Björk's iconic logo. The font only contains A-Z in both lowercase and capitals (so far). However, it does contain alternate glyphs for easier typography and logo making.

The Google Drive link also includes an Illustrator file that shows how Hypik was created and encourages further iteration.
