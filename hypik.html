hypik (font design)

This display font is an homage to Y2K aesthetics and Björk's iconic logo. The font only contains A-Z in both lowercase and capitals (so far). However, it does contain alternate glyphs for easier typography and logo making. I've also included an Illustrator file that shows how it was created and encourages further iteration.

This font is free for personal and commercial use.
